BTS MEMBRES FLYING PHOTOS
===========================

To display real photos of the BTS members flying on the landing page, please place your image files in this folder with the following names:

1. RM:       public/bts/rm.png
2. Jin:      public/bts/jin.png
3. Suga:     public/bts/suga.png
4. J-Hope:   public/bts/jhope.png
5. Jimin:    public/bts/jimin.png
6. V:        public/bts/v.png
7. Jungkook: public/bts/jungkook.png

Note:
- Use transparent PNG images with their backgrounds removed (cutouts) for the best "flying like Superman" visual effect!
- If an image is missing, the website will automatically fall back to a beautiful, glowing vector silhouette.
